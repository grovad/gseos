# GROVAD Smart Estate OS 🏢✨

**GROVAD Smart Estate OS** is a centralized, intelligent digital platform designed to transform estate management by connecting residents, estate managers, realtors, and service providers in one seamless ecosystem. 

Built with a **Clean Minimalism** design philosophy, the platform replaces manual processes with smart, automated, and data-driven operations.

---

## 🚀 Core Features

### 1. Estate Management Hub
*   **Resident Portal:** Register digitally, track property data, and manage occupancy.
*   **Maintenance Hub:** Real-time submission and tracking of maintenance requests (plumbing, electrical, security, etc.).
*   **Estate Announcements:** Instant notifications for estate-wide alerts, emergencies, and events.

### 2. Real Estate Marketplace 🏠
*   **Buy, Sell & Rent:** A comprehensive marketplace for properties within and beyond the estate.
*   **Multi-Role Support:** Specialized dashboards for **Realtors**, **Property Owners**, and **Prospective Buyers**.
*   **Smart Search:** Filter by listing type (Sale, Rent, Lease) and property type (Apartment, Villa, Land, Office).

### 3. Airbnb-Style Short Stay 🛌
*   **Unit Listing:** Residents can list their units for short-term rental.
*   **Booking System:** Real-time availability, price calculation, and booking management.
*   **Stay History:** Track upcoming and past stays with a polished UI.

### 4. Plan Savings Dashboard 💰
*   **Investment Goals:** Clients can set savings targets specifically for property purchases.
*   **Progress Tracking:** Visual indicators and percentage milestones for savings goals.
*   **Fund Management:** Seamlessly add funds and track your journey toward homeownership.

### 5. Role-Based Access Control (RBAC) 🔐
*   **Admin:** Full estate oversight and request management.
*   **Resident:** Personal dashboard for maintenance and short-stay hosting.
*   **Realtor/Owner:** Tools for property listing and marketplace management.
*   **Client:** Focused experience for property search and savings.

---

## 🎨 Design Philosophy: Clean Minimalism
The UI is built on the principles of **intentional whitespace, high-contrast typography, and a refined color palette**.
*   **Primary Colors:** Deep Slate (`#0F172A`), Sky Blue (`#38BDF8`), and Soft Gray (`#F8FAFC`).
*   **Typography:** Bold, tracking-tight headings for an editorial, premium feel.
*   **Interactions:** Smooth transitions powered by Framer Motion.

---

## 🛠️ Tech Stack
*   **Frontend:** React 19, Vite, TypeScript
*   **Styling:** Tailwind CSS 4.0
*   **Backend/Database:** Firebase (Firestore, Authentication)
*   **Animations:** Framer Motion (motion/react)
*   **Icons:** Lucide React

## ⚙️ Setup & Installation

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/your-username/grovad-smart-estate.git
    cd grovad-smart-estate
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Configure Firebase:**
    Create a `firebase-applet-config.json` in the root directory with your Firebase credentials:
    ```json
    {
      "apiKey": "YOUR_API_KEY",
      "authDomain": "YOUR_AUTH_DOMAIN",
      "projectId": "YOUR_PROJECT_ID",
      "appId": "YOUR_APP_ID",
      "firestoreDatabaseId": "(default)"
    }
    ```

4.  **Run the development server:**
    ```bash
    npm run dev
    ```
